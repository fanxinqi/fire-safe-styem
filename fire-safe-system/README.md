# 捷皆系统


## 环境准备

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

### 启动项目

```bash
npm start
```

### 编译项目

```bash
npm run build
```

### 检查格式

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### 测试代码

```bash
npm test
```
